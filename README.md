See it live here: [`https://www.maximilianpetersson.com/`](https://www.maximilianpetersson.com/)

This portfolio is built with **Next.js** using **TypeScript**. The content on the site is managed in **Sanity.io**, and authentication is handled by a **REST API** using **JSON Web Tokens (JWT)** the auth implementation lives in a separate repository which you can find here: [`https://github.com/PeterssonM/spring-stateless-api.git`](https://github.com/PeterssonM/spring-stateless-api.git).
